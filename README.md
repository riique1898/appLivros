# 📱 App Mobile - Livros Favoritos

**Nome:** Henrique Araujo  

---

## 📖 Descrição do Projeto

Este projeto consiste no desenvolvimento de um aplicativo mobile utilizando Ionic Framework com Vue.js 3, com foco exclusivo na construção das interfaces (UI).

O aplicativo simula um sistema de livros favoritos, permitindo que o usuário visualize uma lista de livros e acesse seus detalhes, como autor, descrição e imagem.

Todos os dados são simulados localmente, sem uso de banco de dados ou integração com APIs externas.

---

## 🖥️ Telas Desenvolvidas

- Tela de Login  
- Tela de Cadastro  
- Tela de Recuperação de Senha  
- Página Home (lista de livros)  
- Página de Detalhes do Livro  

---

## ⚙️ Tecnologias Utilizadas

- Ionic Framework  
- Vue.js 3  
- Vue Router  
- JavaScript / TypeScript  
- CSS
